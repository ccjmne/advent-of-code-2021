## Usage

Install:
```shell
npm install
```

Watch-build code:
```shell
npm run dev:build
```

Watch-run built code:
```shell
npm run dev:run          [-y,--year <year>] [-d,--day <day>]
# or
nodemon --quiet dist/run [-y,--year <year>] [-d,--day <day>]
```

Run solution:
```
# or
node dist/run            [-y,--year <year>] [-d,--day <day>]
# or
dist/run.js              [-y,--year <year>] [-d,--day <day>]
```

## Usage

Install:
```shell
npm install
```

Watch-build code:
```shell
npm run dev:build
```

Watch-run built code:
```shell
npm run dev:run          [--year <year>] [--day <day>]
# or
nodemon --quiet dist/run [--year <year>] [--day <day>]
```

Run solution:
```
# or
node dist/run            [--year <year>] [--day <day>]
# or
dist/run.js              [--year <year>] [--day <day>]
```
